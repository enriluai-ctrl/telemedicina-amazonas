# 🏥 Telemedicina Amazonas — Landing comercial

Plataforma de telemedicina offline-first para las 4 RIS de Amazonas (Bagua, Chachapoyas, Condorcanqui, Utcubamba). Lleva atención médica de calidad a zonas rurales Awajún y Wampis sin depender de internet.

## 🔗 Acceso

- **Landing comercial:** [landing.html](./landing.html)
- **Propuesta política (Dr. Grimaldo):** [amazonas.html](./amazonas.html)

## 📊 Datos del proyecto

- 4 RIS (Redes Integradas de Salud) de Amazonas
- PWA offline-first complementaria (no incluida aquí)
- Política de pricing B2G (sin precios públicos)

## 🛠️ Stack

- HTML5 + CSS3 vanilla + JavaScript vanilla
- Tipografía Inter
- Iconos Font Awesome
- Sin frameworks, sin build step
